# amazon-sales-powerbi-analysis
Power BI E-Commerce Sales Dashboard using Amazon Sales Dataset with raw data, cleaned data, and data cleaning workflow.

📊 Amazon Sales Analysis — Power BI Project

This repository contains my end-to-end Power BI project built using an Amazon-style e-commerce sales dataset. The goal of this project was to explore sales trends, understand customer behavior, and build an interactive dashboard that helps answer real business questions.

📁 Project Contents

Amazon-Sales-Dashboard.pbix – The Power BI dashboard file containing all visualizations, measures, and reports used for the Amazon Sales Analysis.

Dataset.xlsx - The cleaned dataset used to build the dashboard.

Dashboard - Overview.mp4	A short video walkthrough of the Power BI dashboard.

README.md - Project documentation.


**🔎 Overview**

This project presents an interactive Power BI dashboard that highlights key sales insights such as revenue trends, order distribution, product performance, and 

category contribution. It includes dynamic filters, slicers, and a changing title based on the selected region and year.



⭐ Project Workflow

Imported raw data into Power BI

Cleaned & transformed data using Power Query

Built DAX measures for KPIs

Designed interactive visuals

Added filters & improved dashboard usability

Tested dashboard with multiple use-cases



📊 **Dashboard Features**

KPI cards showing revenue, orders, quantity sold, and growth rate

Monthly revenue comparison between 2024 vs 2025

Top 5 & bottom 5 revenue months

Revenue by category and payment method

Dynamic title updating by Year and Region

Interactive slicers for Year, Region, and Category

🧠 Key Insights


Revenue shows noticeable seasonal patterns across months

Some categories consistently drive higher sales compared to others

Payment behavior leans strongly toward digital methods

Regional performance varies, highlighting differences in customer behavior
